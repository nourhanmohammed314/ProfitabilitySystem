# Profitability System

## Features

* Product and service management
* Sales transaction tracking
* Monthly fixed cost management
* Variable and fixed cost analysis
* Revenue, total cost, and profit calculation
* Break-even point analysis
* Margin of safety calculation
* Product profitability classification (Strong, Stable, Critical, Loss)
* Product search and reporting
* SQL Server database integration
* Exception handling and input validation

## Technologies Used

* **C++**
* **Object-Oriented Programming (OOP)**
* **Microsoft SQL Server**
* **ODBC (Open Database Connectivity)**
